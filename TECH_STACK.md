# Technology Stack

## Frontend Technologies

- Next.js 15.3.2
- React 19.0.0
- TypeScript 5.8.3
- TailwindCSS 4.1.7

## Backend Technologies

- Node.js 22 LTS
- Next.js API Routes 15.3.2
- TypeScript 5.8.3

## Database Technologies

- PostgreSQL 16
- Prisma ORM 6.8.2

## Deployment Strategy

- Frontend and API routes deployed on Vercel
- PostgreSQL hosted on Neon or an equivalent managed PostgreSQL platform
- Environment variables managed through platform secrets and `.env` files locally
- Prisma migrations applied through the standard Prisma workflow
